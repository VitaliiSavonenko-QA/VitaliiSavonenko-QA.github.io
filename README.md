# Vitalii QA Portfolio

How to use:

1. Unzip the folder.
2. Open `index.html` in your browser.
3. Replace placeholder files in the `assets` folder:
   - photo.jpg — your portrait
   - cv.pdf — your CV
   - bug-report-example.pdf — sanitized bug report example
   - checklist-miro.png — screenshot of a checklist or Miro board
   - test-case-example.png — screenshot or export of a test case example
4. Keep the file names the same, or update paths in `index.html`.

Important:
- Do not move `index.html` away from the `css`, `js`, and `assets` folders.
- The relative paths must stay the same:
  css/style.css
  js/script.js
  assets/photo.jpg
